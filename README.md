## Travel Planner

A simple travel planner application.

## Requirements

* PHP >= 8.0
* Composer >= 2.0.0

## Installation

### Configure the .env file

### Install the dependencies using Composer

```bash
composer install
```

### Migrate the database

```bash
php artisan migrate
```

## Run

```bash
php artisan serve
```

## Test

```bash
php artisan test
```
